# AutoCurseScript智慧树刷课脚本
只能看课不能答题
# How to use

- 在account.txt文件中输入你的账号和密码（记得换行）
- 直接通过Python运行或将项目导入pycharm运行
- 在登录界面手动验证滑块验证
- 选择需要播放的智慧树选修课，点进去即可




# Requirements

- windows PC
- Edge浏览器
- Edge浏览器的驱动（驱动程序必须和main.py在同一目录下）
- python解释器
- 安装selenium包

# Feature

- 不支持过验证码，登录时需要手动过验证码~~奶奶滴网易易盾我记住你了~~
- 可以自动播放还没看完的视频
- 自动关闭播放过程弹出的弹题测验
- 不支持答题，考试（大佬可以尝试添加相应的模块）
- 对于播放过程中随机出现的字母验证也没有办法
